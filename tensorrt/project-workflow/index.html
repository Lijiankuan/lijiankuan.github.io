    参考 sampleMNIST

    • fileData 从 PGM 读取出的数据保存在其中

    • buffers.getHostBuffer 获取 host buffer，将 fileData 复制到 hostbuffer

    • 无法获取 hostbuffer，由于onnx模型无法解析tensorname

    • engine->getNbBindings 获取绑定点数目

    • 假设 binding 0 为 input，1 为 output

    • 将文件读入 0， 拷贝到 device input

    • device output 读回，写入文件

    • 写入/读回都成功，inference 后失败

    数据格式

    A.2. Data Format Descriptions

    参考 sampleFasterRCNN

    float* data = new float[N * INPUT_C * INPUT_H * INPUT_W];
// Pixel mean used by the Faster R-CNN's author
float pixelMean[3]{102.9801f, 115.9465f, 122.7717f}; // Also in BGR order
for (int i = 0, volImg = INPUT_C * INPUT_H * INPUT_W; i < N; ++i)
{
    for (int c = 0; c < INPUT_C; ++c)
    {
        // The color image to input should be in BGR order
        for (unsigned j = 0, volChl = INPUT_H * INPUT_W; j < volChl; ++j)
            data[i * volImg + c * volChl + j] = float(ppms[i].buffer[j * INPUT_C + 2 - c]) - pixelMean[c];
    }
}
