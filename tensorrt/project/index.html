    测试环境

    硬件

    • GeForce RTX 2080

    • i3-4160 CPU @ 3.60Hz

    • 7.94 GB RAM

    软件

    • 显卡驱动 430.39
    • Windows 10 version 1903
    • Cuda 10.0
    • cuDnn 7.5.0
    • TensorRT 5.1

    对比结果

    模型 input batch fp32 fp16 int8
    ResNet50 224x224 41 37ms 10ms 5.7ms
    cnn-10 256x256 31 41ms 24ms 8.7ms
    cnn-10 1080x1920 1 43ms 24ms 8.7ms
    rcnn-10 1080x1920 1 44ms 25ms 8.9ms
