    《机器人学中的状态估计》

    详细描述了三维空间机器人的状态估计，值得一读。

    Chapter 1 状态估计

    • 机器人学中的重要问题：状态估计和控制
    • 状态：一组完整描述他随时间运动的物理量，如位置、速度、角速度

    1.1 状态估计简史

    • 航海定位
    • 天文学是状态估计理论起源
      • 高斯证明了正态分布误差假设下，最小二乘即最优解
    • 卡尔曼
      • 能关性：动态系统的状态何时能从该系统的一组观测值中推断出来
      • 卡尔曼滤波，估计系统状态的优化框架，针对受高斯观测噪声影响的线性系统状态估计方法
    • 新型传感器兴起
      • 相机、激光、GPS

    1.2 传感器、测量和问题定义

    • interoceptive
      • 加速度计、陀螺仪、轮式编码器
    • exteroceptive
      • 相机、ToF（激光、GPS）

    最好的状态估计：同时有效融合内感受型、外感受型传感器观测值

    • 状态估计：根据系统先验模型和测量序列，对系统内在状态进行重构。

    1.3 本书组织结构

    • 状态估计机理

      • 递归、批方法
      • 贝叶斯方法、最大后验估计
      • 连续时间状态估计、高斯过程回归方法
      • 鲁棒估计和偏差

    • 三维空间运动机理

      • 李群李代数
    • 应用
      • 结合前两部分

    2.2.8 高斯分布随机变量的非线性变换

    • p(y) = \int_{-\infty}^{\infty} p(y|x) p(x) \mathrm{d}x，贝叶斯推断分母存在这样的变换
    • p(y|x) = \mathcal{N}(g(x), R)，表示非线性变换后，仍受一个零均值高斯噪声影响，常见于传感器中
    • p(x) = \mathcal{N}(\mu_x, \Sigma_{xx})，理解为状态量，高斯分布

    一般情况下的线性化处理

    • g(x) \approx \mu_y + G(x-\mu_x) 泰勒公式
    • G=\frac{\partial g(x)}{\partial x} \rvert_{x=\mu_x}
    • \mu_y=g(\mu_x)

    Chapter 3 状态估计，Linear Gaussian（LG）

    3.1 离散时间的批估计

    3.1.1 问题定义

    • 运动方程：x_k = A_{k-1}x_{k-1} + v_k + w_k
    • 观测方程：y_k = C_kx_k + n_k

    最大后验估计、贝叶斯推断，两种方法估计系统状态 LG状态估计的根本推导来源于此 批估计，利用全部观测、数据推断当前时刻状态，利用Cholesky分解、RTS平滑算法 递归估计，当前时刻状态仅由之前信息决定。

    关联： 利用贝叶斯公式、马尔可夫性、条件独立， 利用最大后验估计，极值点， 利用运动-观测方程的线性高斯分布， 连立求解出系统状态 利用Cholesky分解，进行批估计，批估计利用所有时刻信息来预测状态 通过边缘化，得到关于某时刻的估计，估计又分为利用k之前信息以及k未来信息 仅利用k之前信息进行估计，得到前向估计，称为滤波，Filter 进一步，k可以利用k-1信息得到结果，得到卡尔曼滤波形式 （MAP、贝叶斯两种推导）
