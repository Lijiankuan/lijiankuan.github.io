    UE4 入门

    源码安装

    • 您可以访问Epic工程师每天开发的最新功能和错误修正，获得真正意义上的最新版本虚幻引擎。
    • 如果您发现了我们尚未修正的错误，而它对您的项目影响又很大，您可以在自己的源代码版本中修正它，并重新编译自己的二进制文件，从而避免耽误项目进度。
    • 您可以将自己的错误修正和自制功能提交给Epic，以改进引擎并帮助整个虚幻社区。

    术语

    • Project
    • Object：基类，提供序列化、垃圾回收等功能
    • Class
    • Actor：可放入关卡中的对象都是 Actor，是支持三维转换（如平移、旋转和缩放）的泛型类
    • Component：不可独立存在，依托于 Actor
    • Pawn 人形体：Actor的子类，可被控制
    • Character 角色：Pawn 的子类，玩家角色
    • PlayerController 玩家控制器：将输入转换为交互，拥有 Pawn/Character
    • AIController 人工智能控制器：与 PlayerController 一样用于控制，拥有 Pawn/Character
    • Brush 画刷：一种 Actor，用于制作关卡，可以附加多种效果，如触发体积、阻塞体积、伤害体积等
    • Level 关卡：游戏区域，用于放置、编辑 Actor，被单独保存为 .umap 文件
    • World 世界：关卡列表
    • GameMode 游戏模式：负责设置游戏规则
    • GameState 游戏状态：游戏分数、玩家人数、其他必要的游戏信息
    • PlayerState 玩家状态：玩家生命值、等级等

    工具和编辑器

    • 关卡编辑器：与上述“Level 关卡”概念对应
    • 材质编辑器：应用于一个模型来控制模型的可见外观
    • 蓝图编辑器：可被当作 Actor 创建，用脚本响应关卡事件
    • 行为树编辑器：可视化的基于节点的脚本，控制 AI 行为
    • Persona 编辑器：动画编辑器
    • 级联粒子编辑器：粒子效果查看、编辑
    • UMG 界面编辑器：创建 UI 界面
    • Matinee 编辑器：过场动画等
    • Sound Cue 编辑器：声音编辑
    • Paper2D 图片编辑器：2D 图片显示
    • Paper2D Flipbook 编辑器：手绘风 2D 动画
    • Physics Asset 工具编辑器：为 骨架网格体 创建物理约束
    • 静态网格体编辑器：对模型外观、碰撞、UV 预览
    • 媒体播放编辑器：编辑媒体文件
